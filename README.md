# Lexo Online Banking

A full-featured online banking system with real-time chat, transfers, and admin panel.

## Features
- User registration & login
- Create PIN for security
- Send/Receive money (internal & external transfers)
- Real-time live chat with support
- Transaction history & notifications
- Admin panel for user management
- Account lock/unlock functionality

## Tech Stack
- Node.js
- Express
- Socket.io (real-time chat)
- HTML/CSS/JavaScript
